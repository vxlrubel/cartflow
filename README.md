# CartFlow

CartFlow is a **modern, scalable, API-first eCommerce platform** built with **Laravel (API backend)** and **Vue.js (Admin Dashboard & Storefront)**.  
It’s designed to handle professional eCommerce needs, including:

- Role-Based Access Control (RBAC): Admin, Manager, Customer
- Full product, inventory, and order management
- Advanced coupon & offer system
- Analytics & reporting dashboards
- Soft delete & trash system for restoring data
- Event-driven architecture with queues and email notifications
- Redis caching and Elasticsearch for fast search
- Activity logs & audit trails

---

## ⚙️ Features

### Admin
- Manage users, roles, and permissions
- CRUD for products, categories, brands, and variations
- Manage orders, payments, and refunds
- Create coupons and promotions
- Access full reports and analytics

### Manager
- Manage products and limited dashboard access
- Process orders
- View reports

### Customer
- Browse products
- Manage profile, wishlist, and orders
